This project is developed in MonoDevelop on Linux, should work as is on Visual Studio and Windows.
All it does is generating png files of Kanjis from kanji.txt and creates configuration file that is used later on the phone to define which file and witch portion of it is kanji image.
It requires Ume Fonts:
http://en.sourceforge.jp/projects/ume-font/

kanji.txt is from:
http://ziggr.com/heisig/
